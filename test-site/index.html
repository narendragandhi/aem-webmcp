What is WebMCP?
WebMCP (Web Model Context Protocol) is a browser API being developed by Google that allows websites to expose structured tools to AI agents.
How does it work?
AI agents can discover and interact with page components through a standardized protocol, enabling reliable automation.
Why use AEM WebMCP?
Automatically enhances AEM Core Components with AI agent capabilities without requiring custom development.
